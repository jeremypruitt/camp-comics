# `STYLE_SUFFIX` is load-bearing — port verbatim, don't trim without cohort regression

The `STYLE_SUFFIX` prompt tail at `_legacy/scripts/generate.py:66–80` is the project's primary anti-drift mechanism. It enforces face-fidelity (to the photo reference, with explicit feature-by-feature language) and costume continuity (to the hero + continuity references), and it places the face-fidelity language at the *end* of the prompt because the model weighs recency. This shape was iterated on through cohort runs that exhibited drift; the current text is the version that produced acceptable identity consistency across 12 panels. The iOS port (`CampComicsCore.PromptBuilder`) must use this string verbatim. Trimming it to "clean up" the prompt without a full-cohort A/B regression is how drift comes back.
